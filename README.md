# MCP Server

This repo records MCPs I have used and I will comment on these MCPs.

## Quick Start

1. Install dependencies
   ```bash
   npm install
   ```
2. Build the project
   ```bash
   npm run build
   ```
3. Start the MCP server
   ```bash
   node build/index.js
   ```
