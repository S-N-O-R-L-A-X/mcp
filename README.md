# MCP TypeScript Server

本项目为基于 TypeScript 的 Model Context Protocol (MCP) 服务端，集成 Playwright。

## 快速开始

1. 安装依赖：
   ```bash
   npm install
   ```
2. 构建项目：
   ```bash
   npm run build
   ```
3. 启动 MCP 服务：
   ```bash
   node build/index.js
   ```

## 参考文档
- MCP 官方文档：https://modelcontextprotocol.io/quickstart/server
- Playwright：https://playwright.dev/
